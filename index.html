Final Machine Learning Project

The team information is placed here only because the final project is the group/team work section.

Early Detection of Parkinson’s Disease Using Supervised Machine Learning

This project uses supervised machine learning to support early detection of Parkinson’s disease using biomedical and clinical features. The main challenge is class imbalance, because medical datasets may contain fewer disease cases than control cases. The project focuses on building models that can detect Parkinson’s cases with strong recall.

Healthcare AI Biomedical Data Class Imbalance Model Evaluation

Problem

Parkinson’s disease can be difficult to detect early because symptoms may be subtle.

Goal

Train machine learning models that can identify Parkinson’s cases accurately and reduce missed positive cases.

Challenge

Class imbalance may make models biased toward the majority class, so recall is important.

Project Workflow

A clear pipeline showing what was done in the machine learning project.

Step 1

Dataset Understanding

Reviewed the dataset, checked the target classes, and understood the meaning of the biomedical features.

Step 2

Preprocessing

Checked missing values, removed irrelevant columns, encoded categorical data when needed, and applied feature scaling.

Step 3

Exploratory Data Analysis

Studied class distribution, feature patterns, correlations, and important visualizations.

Step 4

Model Training

Trained supervised machine learning models such as Logistic Regression, Decision Tree, Random Forest, SVM, KNN, Naive Bayes, and boosting models.

Step 5

Evaluation

Compared models using accuracy, precision, recall, F1-score, AUC-ROC, confusion matrices, and cross-validation.
