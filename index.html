Final Machine Learning Project

Early Detection of Parkinson’s Disease Using Supervised Machine Learning

This project uses supervised machine learning to support early detection of Parkinson’s disease using biomedical and clinical features. The main challenge addressed in this project is class imbalance, because medical datasets may contain fewer disease cases than control cases.

Problem

Parkinson’s disease can be difficult to detect early because symptoms may be subtle.

Goal

Build a machine learning model that can detect Parkinson’s cases with strong recall.

Challenge

Class imbalance can make models miss positive Parkinson’s cases.

Dataset and Preprocessing

The project used Parkinson’s disease datasets containing biomedical and clinical features. Preprocessing was important to prepare the data before model training.

  • Checked missing values
  • Analyzed class distribution
  • Studied feature correlations
  • Removed irrelevant identifier columns
  • Applied encoding when needed
  • Applied feature scaling
  • Used stratified train/test split
  • Applied SMOTE only on the training set to handle imbalance

Why Gradient Boosting Was Selected

Gradient Boosting was selected because it achieved the highest recall, F1-score, and ROC-AUC. Recall is especially important in medical screening because missing a real Parkinson’s case may delay follow-up and treatment.

Key Findings

  • Class imbalance was an important challenge in the dataset.
  • SMOTE improved the model’s ability to detect Parkinson’s cases.
  • Accuracy alone was not enough for evaluating medical screening models.
  • Recall, F1-score, and ROC-AUC were more important for this healthcare task.
  • Gradient Boosting gave the best balance for early Parkinson’s disease screening.
