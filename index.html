Final Machine Learning Project

Early Detection of Parkinson’s Disease Using Supervised Machine Learning

This project uses supervised machine learning to support early detection of Parkinson’s disease using biomedical and clinical features. The main challenge is class imbalance, because disease cases may be fewer than control cases. If imbalance is not handled, the model may look accurate but still miss positive Parkinson’s cases.

Clinical Problem

Parkinson’s symptoms may be subtle in early stages, which can delay diagnosis and follow-up.

Project Goal

Build a machine learning pipeline that detects Parkinson’s cases with strong recall.

Main Challenge

Class imbalance can cause models to favor the majority class and miss disease cases.

Dataset and Preprocessing

The project used Parkinson’s disease datasets containing biomedical and clinical features. Preprocessing was required to prepare the data before training and to reduce bias caused by imbalance.

Missing values were checked and handled.
Class distribution was analyzed.
Feature correlations were studied.
Irrelevant identifier columns were removed.
Feature scaling was applied.
SMOTE was applied only on the training set.

Machine Learning Pipeline

Raw Dataset → EDA → Cleaning → Encoding & Scaling → Stratified Train/Test Split → SMOTE → Model Training → Hyperparameter Tuning → Evaluation → Final Model Selection
SMOTE was applied only after the train/test split to avoid data leakage and keep evaluation realistic.

Models Used

Logistic Regression Support Vector Machine Random Forest K-Nearest Neighbors Naive Bayes Gradient Boosting Ensemble Model

Final Model Results

The final selected model was the Tuned Gradient Boosting Classifier. It was selected because it achieved the strongest medically important results, especially recall, F1-score, and ROC-AUC.

Recall 0.954
F1-score 0.913
ROC-AUC 0.966
Accuracy 0.867

Why Gradient Boosting Was Selected

Gradient Boosting was selected because it achieved the highest recall, F1-score, and ROC-AUC. In medical screening, recall is very important because a false negative means a real Parkinson’s case may be missed. The model also showed strong class separation with ROC-AUC of 0.966.

Key Findings

  • Class imbalance was one of the most important challenges in this project.
  • SMOTE improved the ability of the model to detect Parkinson’s cases.
  • Accuracy alone was not enough for evaluating a healthcare screening model.
  • Recall, F1-score, and ROC-AUC were more meaningful for medical interpretation.
  • Gradient Boosting achieved the best balance for early Parkinson’s disease detection.
