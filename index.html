Final Machine Learning Project

Early Detection of Parkinson’s Disease Using Supervised Machine Learning

This project uses supervised machine learning to support early detection of Parkinson’s disease using biomedical and clinical features. The main challenge is class imbalance because disease cases may be fewer than control cases. If imbalance is not handled correctly, a model may look accurate but still miss real Parkinson’s cases.

Clinical Problem

Parkinson’s disease can be difficult to detect early because symptoms may be gradual and subtle.

Project Goal

Build a supervised ML pipeline that detects Parkinson’s cases with strong recall and ROC-AUC.

Main Challenge

Class imbalance may cause the model to favor the majority class and miss disease cases.

Dataset and Preprocessing

The project used Parkinson’s disease datasets containing biomedical and clinical features. Preprocessing was required to clean the data, prepare features, reduce bias, and support fair model comparison.

Checked missing values
Analyzed class distribution
Studied feature correlations
Removed irrelevant identifier columns
Applied encoding when needed
Applied feature scaling
Used stratified train/test split
Applied SMOTE only on the training set

Machine Learning Pipeline

Raw Dataset → EDA → Cleaning → Encoding & Scaling → Stratified Train/Test Split → SMOTE → Model Training → Hyperparameter Tuning → Evaluation → Final Model Selection
SMOTE was applied only after the train/test split and only on the training set to prevent data leakage.

Models Used

Logistic Regression Support Vector Machine Random Forest K-Nearest Neighbors Naive Bayes Gradient Boosting Ensemble Model

Final Model Results

The final selected model was the Tuned Gradient Boosting Classifier. It achieved the strongest medically important results, especially recall, F1-score, and ROC-AUC.

Recall 0.954
F1-score 0.913
ROC-AUC 0.966
Accuracy 0.867

Why Gradient Boosting Was Selected

Gradient Boosting was selected because it achieved the highest recall, F1-score, and ROC-AUC. In medical screening, recall is very important because a false negative means a real Parkinson’s case may be missed. The model also showed strong class separation with ROC-AUC of 0.966.

Key Findings

  • Class imbalance was one of the most important challenges in this project.
  • SMOTE improved the ability of the model to detect Parkinson’s cases.
  • Accuracy alone was not enough for evaluating a healthcare screening model.
  • Recall, F1-score, and ROC-AUC were more meaningful for medical interpretation.
  • Gradient Boosting achieved the best balance for early Parkinson’s disease detection.
